本站已添加RSS订阅链接

2026年02月22日 701 字 大概 2 分钟

通过Hexo插件，本站已添加RSS订阅链接，可以导入到各个平台的RSS阅读器中，既免去了阅读文章需要进入网站的麻烦，而且能够通过客户端及时收到网站文章更新，十分方便。

如何将RSS链接导入到客户端

下载RSS阅读器

目前网上RSS阅读器很多，个人比较喜欢的是Feedly，不过可能需要科学上网。目前iOS端在用Ego Reader。

添加链接

在客户端内找到添加链接的选项，然后输入“https://www.asutaka.ink/atom.xml”
客户端会自动识别网站的基本信息，当然自己也可以修改。添加完成后客户端应该会显示本站所有文章。

打开通知

手机端的RSS阅读器应该都有推送通知，只需要在手机里打开通知权限即可。

如何实现Hexo博客生成RSS链接

适用于使用Hexo开设个人博客的站长们，实测Github Page也可以实现

本内容来源自腾讯云开发者社区：为Hexo项目添加RSS订阅链接

  1. 首先添加功能插件，在 hexo 项目根目录下执行该命令：
    npm install hexo-generator-feed --save
    注意：（Linux、macOS可能需要管理员权限，即在命令前加上sudo）

  2. 然后在 hexo 根目录下的_config.yml文件中添加配置

    配置含义：:
    type: RSS的类型(atom/rss2)
    path: 文件路径，默认是 atom.xml/rss2.xml
    limit: 展示文章的数量,使用 0 或则 false 代表展示全部
    hub: URL of the PubSubHubbub hubs (如果使用不到可以为空)
    content: （可选）设置 true 可以在 RSS 文件中包含文章全部内容，默认：false
    content_limit: （可选）摘要中使用的帖子内容的默认长度。 仅在内容设置为false且未显示自定义帖子描述时才使用。
    content_limit_delim: （可选）如果content_limit用于缩短post内容，则仅在此分隔符的最后一次出现时进行剪切，然后才达到字符限制。默认不使用。
    icon: （可选）自定义订阅图标，默认设置为主配置中指定的图标。
    order_by: 订阅内容的顺序。 (默认: -date)

  3. 然后在 theme 目录下的_config.yml文件中添加配置：
    rss: /atom.xml
    注意：有的Hexo主题配置文件同样在根目录，只需要记得第二步修改的是Hexo配置文件，这一步修改的是主题配置文件

  4. 随后重新生成博客静态文件
    hexo clean && hexo g（其实直接输入hexo generate也可以），可能需要管理员权限

  5. 在 public 文件夹中就会生成 atom.xml 文件，部署后直接在根目录中访问该文件即可
